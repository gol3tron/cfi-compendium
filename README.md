## ğŸš€ Getting Started

### Viewing the Site Locally

1. Clone the repository:
```bash
git clone https://github.com/gol3tron/cfi-compendium.git
cd cfi-compendium
```

2. Open `index.html` in your web browser, or serve it with a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Or simply open index.html directly in your browser
```

3. Navigate to `http://localhost:8000` (or the appropriate URL)

### Browsing Content

- Use the **sidebar navigation** to browse lesson categories
- Use the **search bar** to find specific topics
- Click on any lesson to view the full content
- All lessons are written in Markdown for easy reading and editing
